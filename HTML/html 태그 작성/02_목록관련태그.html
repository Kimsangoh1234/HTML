o1 Ordered List 순서가 있는 목록

      type 속성 : 표시되는 숫자 모양 변경
      type "a"  : 영어소문자 시작(a,b,c,d... )
      type "A"  :  영어대문자 시작(A,B,C,D,..)
      type "i"  :  로마 소문자(i,ii,iii,iiii,..)
      type "I"  :  로마 대문자 시작(I,II, III,IIII,..)
      type "1"  :  숫자 시작(1,2,3,4,5..)

      start "속성" :시작하는 숫자 지점
      reversed : 거꾸로 시작
